Project Title: SAP MM Procure-to-Pay (P2P) Cycle Automation Dashboard

Name: Sastika Deb
Roll No: 2307077
Module: SAP MM (Procure-to-Pay Cycle)

Description:
This project demonstrates the end-to-end Procure-to-Pay (P2P) lifecycle
implementation using a simulated SAP MM environment for Zenith Industries Ltd.

The system includes Vendor Master, Material Master, Purchase Requisition,
Purchase Order, Goods Receipt, Invoice Verification, and Payment Processing.

Features:
• Real SAP-style document numbering
• 3-way matching (PO–GR–Invoice)
• Dashboard KPIs and reports
• ALV-style export functionality
• Fiori-inspired UI design

Tools Used:
Frontend: React, TailwindCSS
Backend: FastAPI (Python)
Database: MongoDB
Version Control: GitHub
